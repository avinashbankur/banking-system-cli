🏦 Python Banking System (CLI-Based)

A simple Command Line Interface (CLI) banking system built using Python.  
This project simulates basic banking operations such as login authentication, balance management, deposits, and withdrawals.

🚀 Features

User Login Authentication
Deposit Money
Withdraw Money
Check Account Details
Change Account Type
Account Freeze after multiple failed attempts

🧠 Concepts Used

Object-Oriented Programming (OOP)
Classes & Objects
Conditional Statements
Loops
User Input Handling


📂 Project Structure

How It Works

1. User enters username
2. System verifies account existence
3. Login using:
   - Account Number
   - Username
   - Password (max 3 attempts)
4. After login, user can:
   - View details
   - Deposit money
   - Withdraw money
   - Change account type
